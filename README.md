# System Design — Hands-On in Java

Learning system design by building it. Every topic here ends as one of:
**runnable Java code**, or a **Mermaid diagram + written tradeoff analysis** I constructed —
never just notes from watching videos.

## How this repo works
- `01-foundations/` → SOLID, CAP, DB internals — with code demos
- `02-patterns/` → design patterns, learned just-in-time as real problems demand them
- `03-lld/` → full low-level designs in Java (Parking Lot, Elevator, BookMyShow, Splitwise…)
  — each with a class diagram matching the actual code
- `04-estimation/` · `05-building-blocks/` → HLD primitives (rate limiter, consistent hashing,
  caching, load balancing) — each proven with a minimal runnable implementation
- `06-microservices/` · `07-security/` → distributed patterns, auth, encryption
